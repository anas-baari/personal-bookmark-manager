# 🔖 Professional Bookmark Manager
> A functional web application for categorizing and storing digital resources.

## 🚀 Why this matters:
This isn't just a static website—it's a **Data-Driven Application**. It demonstrates my ability to handle user input, manage local databases, and create a seamless "Single Page" experience.

## 🛠️ Core Features
* **Persistent Storage:** Uses `localStorage` so your bookmarks stay saved even after you close the browser or restart your computer.
* **Category Management:** Users can organize links into News, Work, Entertainment, or Miscellaneous.
* **Full CRUD Logic:** Includes functionality to **Create** new bookmarks, **Read** (filter by category), and **Delete** entries.
* **Responsive UI:** Styled with a professional dark-mode theme that is easy on the eyes and optimized for mobile devices.

## 🧠 Technical Skills Demonstrated
* **State Management:** Handling complex app states (showing/hiding sections) using JavaScript DOM manipulation.
* **Data Validation:** Ensuring users provide valid names and URLs before saving data.
* **Modern CSS:** Implementing CSS Variables and Flexbox for a clean, maintainable codebase.
* **Clean Code:** Organized into logical functions like `getBookmarks()`, `displayBookmarks()`, and `deleteBookmark()`.

## 📂 Project Structure
* `index.html` - Semantic HTML structure.
* `styles.css` - Custom UI with a golden-yellow and dark-grey professional palette.
* `script.js` - The engine that handles all data logic and user interactions.
